1Ô∏è‚É£ What is a Favicon?

A favicon (favorite icon) is a small image displayed in multiple places:

  • Browser Tab: Appears next to the page title in the browser tab
  • Bookmarks Bar: Shows in browser bookmarks
  • History: Visible in browsing history
  • Address Bar: Displays in the address bar/URL bar
  • Home Screen: On smartphones when added to home screen
  • Browser Tabs: When multiple tabs are open

Example:

When you open Google, Amazon, or Facebook, you'll see their branded icon in the browser tab. That's their favicon!

3Ô∏è‚É£ Implementation Methods (7 Ways)

Method 1: Traditional ICO Format (Simplest)

Place a file named favicon.ico in your website root directory. No HTML tag needed!

HTML
<!-- Simply place favicon.ico in root directory -->

How it works: Browsers automatically look for favicon.ico in the root directory.

Method 2: Using <link> Tag (Most Common)

Explicitly specify the favicon using the <link> tag in the <head> section.

HTML
<!-- Single favicon --> <link rel="icon" type="image/x-icon" href="favicon.ico"> <!-- Multiple sizes (modern approach) --> <link rel="icon" type="image/png" sizes="32x32" href="favicon-32x32.png"> <link rel="icon" type="image/png" sizes="16x16" href="favicon-16x16.png">

Method 3: SVG Favicon (Modern & Scalable)

SVG favicons are infinitely scalable, perfect for modern browsers.

HTML
<link rel="icon" type="image/svg+xml" href="favicon.svg">

Method 4: Apple Touch Icon (iPhone/iPad)

When users add your site to their home screen on Apple devices.

HTML
<link rel="apple-touch-icon" href="apple-touch-icon.png"> <link rel="apple-touch-icon" sizes="180x180" href="apple-touch-icon-180.png"> <link rel="apple-touch-icon" sizes="152x152" href="apple-touch-icon-152.png">

Method 5: Microsoft Tile Image (Windows)

When website is pinned to Windows Start menu.

HTML
<meta name="msapplication-TileImage" content="mstile-144x144.png"> <meta name="msapplication-TileColor" content="#1f5db7">

Method 6: Android Chrome Icon (Web App Manifest)

For Android devices and PWA (Progressive Web Apps).

HTML
<link rel="manifest" href="manifest.json">
manifest.json
{ "icons": [ { "src": "favicon-192x192.png", "sizes": "192x192", "type": "image/png" }, { "src": "favicon-512x512.png", "sizes": "512x512", "type": "image/png" } ] }

Method 7: Theme Color & Meta Tags

Specify theme colors that browsers use for UI elements.

HTML
<meta name="theme-color" content="#1f5db7"> <meta name="msapplication-navbutton-color" content="#1f5db7">

6Ô∏è‚É£ Common Mistakes to Avoid

‚ùå Mistake 1: Wrong File Path

WRONG
<link rel="icon" href="images/favicon.png"> <!-- Favicon not found -->
RIGHT
<link rel="icon" href="/favicon.png"> <!-- Use absolute path -->

‚ùå Mistake 2: Missing Type Attribute

WRONG
<link rel="icon" href="favicon.png"> <!-- Browser might try wrong format -->
RIGHT
<link rel="icon" type="image/png" href="favicon.png"> <!-- Explicitly specify type -->

‚ùå Mistake 3: Too Large File Size

WRONG
favicon.png (5MB) <!-- Way too large! -->
RIGHT
favicon.png (5-10KB) <!-- Optimized -->

‚ùå Mistake 4: Poor Quality Design

At 16x16 or 32x32 pixels, your favicon needs to be bold and simple. Avoid thin lines and small details that disappear at small sizes.

‚ùå Mistake 5: Not Clearing Cache

After updating your favicon, browsers cache the old one. Hard refresh (Ctrl+F5) to see changes immediately.

7Ô∏è‚É£ Favicon Design Best Practices

  • Keep it simple: A favicon should be recognizable at 16x16 pixels
  • Use bold colors: Avoid gradients and subtle shading that fade at small sizes
  • Make it square: Most favicons are square (16x16, 32x32, etc.)
  • Brand consistency: Use your company's logo or primary color
  • Contrast matters: Ensure contrast between icon and background
  • Test at small sizes: Always preview your favicon at actual sizes
  • Consider dark mode: Make sure your favicon is visible on dark browser themes
  • Use geometric shapes: Circles, squares, and triangles work best
  • Limit colors: 2-4 colors work best; too many become muddy
